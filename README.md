# QuizFlow

**Application de gestion de quiz, devoirs et gamification pédagogique**

Une plateforme interactive développée avec **Flutter** et **Firebase** permettant aux enseignants de créer des évaluations variées et aux étudiants de progresser de manière ludique.

![QuizFlow Banner](https://via.placeholder.com/800x300/4A43EC/FFFFFF?text=QuizFlow)

---

## Fonctionnalités Principales

###  Espace Enseignant
- Création de quiz multi-formats (QCM, Vrai/Faux, Matching, Ordre de mots)
- Génération intelligente de questions avec **Gemini AI**
- Gestion des devoirs et suivi des rendus
- Statistiques avancées et tableau de bord
- Notifications en temps réel

###  Espace Étudiant
- Rejoindre un quiz via code d’accès
- Passation interactive avec système **anti-triche**
- Gamification complète (Points, Badges, Streak, Classement)
- Historique des résultats
- Apprentissage autonome avec l’IA

---

##  Technologies

- **Frontend** : Flutter (Dart)
- **Backend** : Firebase (Firestore, Authentication)
- **IA** : Gemini (Google AI)
- **Mobile** : Android Studio (APK 39.9 MB)

## Comment Lancer le Projet

```bash
# Cloner le repository
git clone https://github.com/kz18-vs/quizflow-app.git

# Accéder au dossier
cd quizflow-app

# Installer les dépendances
flutter pub get

# Lancer sur Chrome
flutter run -d chrome

# Lancer sur Android (émulateur ou téléphone)
flutter run

Structure Principale
lib/screens/ → Interfaces (Enseignant & Étudiant)
lib/services/ → Gamification, IA, Notifications
lib/widgets/ → Composants réutilisables

Points Forts
4 types de questions interactives
Système anti-triche intelligent
Gamification complète avec 12 badges
Notifications contextuelles
Version Android compilée avec succès
